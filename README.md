# Stockly

Веб-приложение для прогнозирования спроса на товары розничного магазина на основе исторических данных о движении товаров.  
Проект предназначен для анализа продаж и формирования прогнозов спроса, которые могут использоваться для оптимизации закупок и ценообразования.

### Запуск отладочного сервера под Windows
Используемая версия Python — 3.10.64

1. После клонирования проекта в локальный репозиторий, необходимо развернуть виртуальное окружение:

python -m venv venv

2. После чего стоит активировать его командой:

source venv/Scripts/activate

3. Виртуальное окружение активировано, а значит пора устанавливать зависимости:

pip install -r requirements.txt

4. Описанные в файле зависимости установились в окружении, теперь можно запускать отладочный сервер:

python manage.py runserver
