3. Краткий концепт проекта
Веб-приложение для прогнозирования спроса по товарам розничного магазина на основе данных о движении товаров за 4 года. Позже на основе прогноза добавим автоматическое создание заявки поставщику и рекомендацию цены продажи на основе анализа спроса.
Данные получаются через GraphQL и включают: приход, продажи, списания, инвентаризацию, закупочную и продажную цену, дату операции.
Система формирует прогноз продаж по каждому товару на завтра, неделю, 14 дней и 30 дней вперёд.
Результат отображается в виде таблиц и графиков.
