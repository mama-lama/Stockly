# Stockly

Веб-приложение для прогнозирования спроса на товары розничного магазина на основе исторических данных о движении товаров.  
Проект предназначен для анализа продаж и формирования прогнозов спроса, которые могут использоваться для оптимизации закупок и ценообразования.

## Запуск отладочного сервера
Используемая версия Python — 3.10.64

### 1. Клонировать репозиторий:
git clone https://github.com/mama-lama/Stockly.git

### 2. Создать виртуальное окружение:
python -m venv venv

### 3. Активировать виртуальное окружение:
Для Linux/macOS:
source venv/bin/activate

Для Windows:
source venv/Scripts/activate

### 4. Установка зависимостей:
pip install -r requirements.txt

### 5. Скопировать .env.example в .env:
cp .env.example .env

### 6. Отредактировать .env под свои локальные настройки
nano .env  # или любой редактор

### 7. Запустить миграции
python manage.py migrate

### 8. Запустить сервер
python manage.py runserver
